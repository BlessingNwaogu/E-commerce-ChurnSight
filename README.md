# E-commerce ChurnSight 📉

A Power BI analytics project exploring customer churn patterns in an e-commerce business, built on a Kaggle e-commerce dataset — from raw data to a branded, decision-ready dashboard.

## 🎯 Why churn?

Customer churn is one of the costliest problems in e-commerce — acquiring a new customer typically costs far more than retaining an existing one. This project turns raw customer data into a dashboard that helps a business quickly answer: *how many customers are we losing, and is it getting better or worse?*

## 📊 What's inside

The dashboard focuses on **overview-level KPIs and churn rate trends**, including:
- Overall churn rate and total churned customers
- Churn trend over time
- Key metrics at a glance for quick decision-making

## 🛠️ Built with
- **Power BI** — data modeling, DAX measures, and report design
- **Power Query** — data cleaning and transformation
- Dataset: Kaggle e-commerce customer dataset

## 📁 What's in this repo
